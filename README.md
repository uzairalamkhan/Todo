 ## Todo App

This is a basic to do app in the form of table. The table contains six columns containing Timestamp,Title,Description,Duedate,Tag,Status and action.
The user can add and edit the task and set a desirable due date to it.

## Installation

### 1. clone the repository
you can clone the repository by using the command below

git clone git@github.com:uzairalamkhan/TodoApp.git

### 2. Set the directory 

change the directory to Apps directory

cd TodoApp

### 3.Install Dependencies

npm install antd

npm install moment --save
